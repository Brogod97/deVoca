## 🐝 프로젝트 소개

`개발자 전용` `단어장` `KH D클래스 A11` `세미 프로젝트`

- deVoca는 개발자라는 뜻의 영단어 **dev**eloper와 용어라는 뜻의 **Voca**bulary를 조합하여 개발자 단어장이라는 뜻을 담고 있습니다.

- 보다 개발자 학습에 특화된 형태의 디지털 단어장 서비스입니다.
